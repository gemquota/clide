# CLIDE Documentation Index

Welcome to the **Source of Truth** for CLIDE (Command Line Interface - Database: Everything) v0.4.1.

## Project Thesis
CLIDE is an **Autonomous Agentic Asset Forge**. It solves the "Ephemeral Intent Gap" by monitoring user interactions and shell history to formalize them into permanent, hot-swappable agentic assets (Prompts and MCP Servers).
