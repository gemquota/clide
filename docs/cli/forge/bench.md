# FORGE // BENCH

Executes performance and latency benchmarks for a forged asset.

## Usage
`./cli forge bench <id>`

## Description
This command runs an execution profile on a specific tool or module, measuring average latency and responsiveness. The results are stored in the neural registry to inform the agent's choice of tools based on efficiency.

Title: BENCHMARK METRICS
Primary: Average Latency (seconds)
Telemetry: Execution Profile
Storage: Updated in memory.db
