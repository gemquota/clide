# CLIDE // BRAIN DOMAIN

## Tier: Basic
- query <text> : Ask natural language questions to your project knowledge.
- graph        : Generate visual topology data for the Galaxy view.
- analyze      : Retroactively process history for missed facts.
- summary      : Generate an executive high-level overview.
- audit        : Identify logical contradictions in the DB.
